# Preempting-Text-Sanitization
This repository contains the codebase for the experiments of the paper titled "Preempting Text Sanitization Utility in Resource-Constrained Privacy-Preserving LLM Interactions".

## How to use
The codebase is split into two parts: 
1. The folder "1-Summarization" contains the text summarization experiments of Sections 4.1 to 4.4 included.
2. The folder "2-dx-semantic" contains the code for Section 4.5 on the replication of experiments on $d_\mathcal{X}$-privacy for text.

Please refer to the README.md of each folder for further information.
